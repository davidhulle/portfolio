5. NAVBAR MOBILE — Header e menu
--------------------------------------------------------------
a) Alinhamento do header mobile:
   - Outer padding: '16px 24px 0' (igual ao header fixo).
   - Inner padding: '16px 16px 16px 24px'.
   - LangSelect size="md": height auto, padding '8px 12px'.

b) Animações do menu mobile:
   - Ícone hamburger → X: morph de path SVG via Framer Motion
     (AnimatedMenuIcon com motion.path animando atributo d).
   - Abertura: wave clip-path circle(0 → 150vmax) partindo
     do centro do botão hamburger.
   - Fechamento: wave reversa com ease deceleration.
   - Itens do menu: stagger de entrada (delay 0.22 + i×0.055s)
     e stagger reverso de saída (último sai primeiro).

   - getBoundingClientRect() captura o centro exato do botão
     hamburger no momento do clique.
   - waveOrigin state armazena '${x}px ${y}px' para usar nas
     três propriedades clipPath (initial / animate / exit).
